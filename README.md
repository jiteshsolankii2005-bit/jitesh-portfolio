# Jitesh Solanki Portfolio

This is a static one-page portfolio website for Assistant Accountant job hunting and focused freelance accounting support.

## Local preview

Open `index.html` in a browser, or serve the folder with any static server.

## Deploy on Vercel

The site is deployed on Vercel from the GitHub repository.

Live site: https://jitesh-portfolio-zeta.vercel.app/

Future updates can be made locally, pushed to GitHub, and Vercel will publish the new version automatically.
