# Jitesh Solanki Portfolio

This is a static one-page portfolio website for job hunting and freelance accounting support.

## Local preview

Open `index.html` in a browser, or serve the folder with any static server.

## Deploy on Vercel

1. Create a new GitHub repository.
2. Upload the full `jitesh-portfolio` folder contents.
3. Go to Vercel and choose **Add New Project**.
4. Import the GitHub repository.
5. Keep the framework preset as **Other** or **Static**.
6. Deploy.

Vercel will create a free live link. A custom domain can be connected later from the Vercel project settings.
